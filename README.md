# node_crud_app

With Nodejs, ExpressJs and MongoDB Atlas


. API END POINT

POST   ( '/post)   - create new post
GET.  (/post/:id). - to get a single post 
PATCH.  ('/post/:id).  - edit detail of a single post 
DELETE.  ('/post/:id).  - delete a post
